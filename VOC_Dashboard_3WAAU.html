VOC 대시보드 연결

Google Sheets의 VOC 데이터를 실시간으로 연동합니다.
고객 개인정보(휴대폰·주소)는 자동으로 마스킹됩니다.

📌 사전 준비 (최초 1회)
1. Google Cloud Console에서 프로젝트 생성
2. Google Sheets API 활성화
3. 사용자 인증 정보 → OAuth 2.0 클라이언트 ID (웹 애플리케이션) 생성
4. 승인된 JS 출처에 이 HTML 파일이 열리는 URL 추가
5. Client ID 복사 후 아래 입력
▸ 열 이름 직접 지정 (기본값과 다를 경우)
